# fogair
A Work-In-Progress Rogue-like game in C

![screenshot](assets/screenshot.png)

# Build

Prerequisites:
1. Download [raylib 5.5](https://github.com/raysan5/raylib/releases/tag/5.5) and extract in project directory. 
2. Any C compiler

Run:
```
mkdir -p build/linux
cc -o Buildfile Buildfile.c && ./Buildfile
```

# Run game

Run the executable:
```
./build/linux/exe
```
